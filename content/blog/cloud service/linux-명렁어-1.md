# 📃 Ubuntu란

> 💡 리눅스 커널을 기반으로 한 리눅스 배포판 중 하나.
> 따라서 우분투와 리눅스는 계층 구조가 같다.

최상단에 root 디렉토리가 위치하고 밑에 bin, etc, usr, boot, home 디렉토리가 위치한다.

# 🔍 명령어

| 값      |             의미              |             사용 방법 |
| ------- | :---------------------------: | --------------------: |
| `pwd`   |    현재 작업 디렉토리 확인    |                   pwd |
| `ls`    | 현재 디렉토리 내의 파일 확인  |                    ls |
| `mkdir` |       새 디렉토리 생성        |       mkdir file_name |
| `touch` |         새 파일 생성          |       touch file_name |
| `vi`    |         텍스트 에디터         |          vi file_name |
| `cat`   |        파일 내용 출력         |         cat file_name |
| `man`   |    해당 명령어 사용법 출력    |                man ls |
| `cd`    |         디렉토리 이동         |           cd dir_name |
| `rm`    |           파일 삭제           |          rm file_name |
| `rmdir` |         디렉토리 삭제         |        rmdir dir_name |
| `mv`    | 파일/디렉토리 이동, 이름 변경 | mv file_name dir_name |
